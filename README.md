# Kawoosa Khalisa Football Academy (KKFA)

A production-ready, full-stack football academy platform built with a futuristic cinematic identity.

## 🚀 Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS, Framer Motion, Zustand, Recharts, Lucide Icons.
- **Backend**: Node.js, Express.js, Sequelize ORM (MySQL), JWT.
- **Features**: Multi-wing support (Kawoosa & Kunzer), Admin Dashboard, holographic player profiles, live fixtures, event registrations with PDF generation.

## 🛠️ Installation

### 1. Database Setup
- Create a MySQL database named `kkfa_db`.
- Update `server/.env` with your MySQL credentials.

### 2. Backend Initialization
```bash
cd server
npm install
npm run dev
```

### 3. Frontend Initialization
```bash
cd client
npm install
npm run dev
```

## 🏗️ Folder Structure
- `/client`: React application.
- `/server`: Node/Express API with Sequelize models.
- `/client/src/components`: UI components organized by module.
- `/client/src/pages`: Cinematic page implementations.
